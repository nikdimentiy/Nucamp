Selection Sort

Selection sort is a simple comparison-based sorting algorithm. It repeatedly selects the smallest element from the unsorted region and swaps it with the first element of the unsorted region.

Time Complexity (Big O): O(n^2) - Quadratic time complexity. Space Complexity: O(1) - Constant space complexity.
